┌─────────────────────────────────────────────────────────────────┐
│ 1. ADIM — GITHUB HESABI VE GIT KURULUMU                         │
└─────────────────────────────────────────────────────────────────┘

A) GitHub hesabın yoksa: https://github.com/signup
   (Ücretsiz, sadece email + kullanıcı adı.)

B) Git Windows kurulu mu kontrol et — PowerShell aç:
       git --version
   Eğer 'tanınmıyor' diyorsa: https://git-scm.com/download/win
   (varsayılan ayarlarla kur)

C) Git'e kimliğini söyle (PowerShell'de):
       git config --global user.name "Senin Adin"
       git config --global user.email "senin@email.com"


┌─────────────────────────────────────────────────────────────────┐
│ 2. ADIM — GITHUB'DA PRIVATE REPO OLUŞTUR                        │
└─────────────────────────────────────────────────────────────────┘

A) https://github.com/new aç
B) Doldur:
       Repository name : ott-bot
       Description     : OTT Bot Dashboard
       Visibility      : ☑ Private (sadece sen göreceksin)
       Add README      : ☐ (Boş bırak)
       .gitignore      : ☐ (Boş bırak, kendimiz hazırladık)
C) "Create repository" butonu


┌─────────────────────────────────────────────────────────────────┐
│ 3. ADIM — KODU REPO'YA YÜKLE                                    │
└─────────────────────────────────────────────────────────────────┘

PowerShell'de bu komutları sırayla çalıştır
(her satırdan sonra Enter):

   cd C:\Users\furka\Desktop\ott_bot

   git init
   git branch -M main
   git add .
   git commit -m "ilk commit"
   git remote add origin https://github.com/KULLANICI_ADIN/ott-bot.git
   git push -u origin main

(KULLANICI_ADIN yerini kendi GitHub kullanıcı adınla değiştir!)

İlk push'da kullanıcı/şifre/token isteyebilir. GitHub artık şifre kabul
etmiyor, "Personal Access Token" kullanman gerekir:
   - GitHub → Settings → Developer settings → Personal access tokens
   - "Generate new token (classic)" → repo scope'unu seç → token kopyala
   - Push komutunda şifre yerine token'ı yapıştır

NOT: .env dosyası .gitignore'da, yüklenmez (TradingView şifren güvende).


┌─────────────────────────────────────────────────────────────────┐
│ 4. ADIM — GITHUB ACTIONS İÇİN SECRETS                           │
└─────────────────────────────────────────────────────────────────┘

A) Repo'na git: https://github.com/KULLANICI_ADIN/ott-bot
B) Settings → Secrets and variables → Actions
C) "New repository secret" — iki secret ekle:
       Name: TV_USERNAME      Value: furkan_ylmaz28
       Name: TV_PASSWORD      Value: 626865fyGS**

(Bu bilgiler GitHub Actions'ın haftalık optimize çalıştırırken
TradingView'a giriş yapması için. Repo private + secrets şifreli.)

D) Test et: Actions sekmesine git → "Weekly Optimize" workflow'unu seç
   → "Run workflow" → "Run workflow" butonu
   → ~30 dk sonra başarıyla tamamlanmalı + per_symbol_params.json
     güncellenmiş olarak push edilmeli.


┌─────────────────────────────────────────────────────────────────┐
│ 5. ADIM — STREAMLIT CLOUD'A DEPLOY                              │
└─────────────────────────────────────────────────────────────────┘

A) https://share.streamlit.io açıp GitHub hesabınla giriş yap
   (ilk girişte GitHub bağlantısı yetkileri verirsin)

B) "New app" butonu

C) Doldur:
       Repository       : KULLANICI_ADIN/ott-bot
       Branch           : main
       Main file path   : app.py
       App URL          : ott-bot-furkan   (istediğin alt-domain — boştaysa)

D) "Advanced settings" → Python version: 3.11

E) "Secrets" alanına yapıştır (TradingView bilgilerini):
       TV_USERNAME = "furkan_ylmaz28"
       TV_PASSWORD = "626865fyGS**"

F) "Deploy!" tıkla. İlk build ~5-8 dakika sürer
   (pip install + numba derleme).

G) Tamam! URL'in hazır:
       https://ott-bot-furkan.streamlit.app  (örnek)


┌─────────────────────────────────────────────────────────────────┐
│ 6. ADIM — UPTIMEROBOT İLE UYANIK TUT                            │
└─────────────────────────────────────────────────────────────────┘

A) https://uptimerobot.com — ücretsiz hesap aç (50 monitor)
B) "Add New Monitor":
       Monitor Type        : HTTP(s)
       Friendly Name       : OTT Bot
       URL                 : https://ott-bot-furkan.streamlit.app
       Monitoring Interval : 5 minutes
C) Create Monitor.

Bot bundan sonra hiç uyumaz, her 5 dk'da bir ping atılır.
Eğer bot çökerse email'inle uyarı gelir.


┌─────────────────────────────────────────────────────────────────┐
│ 7. ADIM — ANDROID TELEFONA APP GİBİ EKLE (PWA)                  │
└─────────────────────────────────────────────────────────────────┘

A) Android telefonda Chrome'u aç
B) URL'ini gir: https://ott-bot-furkan.streamlit.app
C) Üst sağ köşedeki üç nokta menüsü → "Install app" (veya "Add to Home screen")
D) "Yükle" — ev ekranında OTT Bot ikonu belirir
E) İkona tıkla → tam ekran, app gibi açılır

iPhone için:
A) Safari'de URL'ini aç
B) Aşağıdaki paylaş ikonu → "Add to Home Screen"
